Linux CLI Quiz – Mini-Lernplattform
======================================

Dieses ZIP enthält eine kleine, eigenständige HTML-Seite mit Quizfunktion
für ausgewählte Linux-Befehle (pwd, cd, cd .., ls, grep, Pipe, Umleitung,
Nano-Workflow).

Dateien
-------
- index.html
  Öffne diese Datei im Browser, um die Quiz-Ansicht zu nutzen.
  Jeder Befehl ist eine Karte mit einem 3-Fragen-Quiz
  (leicht · mittel · schwer). Bei 3/3 richtigen Antworten
  wird die Karte visuell als „Quiz bestanden“ hervorgehoben.

Nutzung in Canva
----------------
- Du kannst die Fragen/Antworten aus der Datei direkt in Canva
  übernehmen (z.B. als Kartenset oder Slides):
  - Für jede Karte: Befehlstitel + 3 Fragen + Antwortmöglichkeiten.
  - Die Fragen sind bereits nach Schwierigkeitsgrad sortiert.
- Alternativ kannst du einen Screenshot / Export der Seite nutzen
  oder das Layout als Referenz für dein Canva-Design nehmen.

Erweiterung
-----------
- Weitere Befehle kannst du nach demselben Muster ergänzen:
  - Einen neuen Eintrag im Objekt `quizData` (im <script>-Block).
  - Eine neue `.command-card` mit passender `data-quiz-id`.

Viel Erfolg beim Lernen!
